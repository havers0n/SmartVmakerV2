## 📞 Если что-то не работает

### ❌ Ошибка "Table does not exist"
**Решение:** Выполни SQL миграции (шаг 1)

### ❌ 404 на `/api/videos`
**Решение:** Проверь что Dashboard запущен на `localhost:3004`

### ❌ Пустой список видео в UI
**Это нормально!** Таблица пуста, пока не добавишь данные:
- Либо выполни `seed.sql` для тестовых данных
- Либо добавь видео вручную через SQL INSERT

### ❌ Ошибка подключения к БД
**Решение:** Проверь `DATABASE_URL` в `.env` файле
